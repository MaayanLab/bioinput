Bioinput is a JavaScript library built around the jQuery, typeahead.js, and Bootstrap Tags Input libraries. It allows you to quickly and easily implement an autocomplete html input element for various biological entities.

Getting Started

Include Bioinput (and its dependencies) by adding them to an HTML page in your project. This is done by first including the a copy of the Bioinput css in your <head></head> tags: 

            
<head>
  ...
  <link rel="stylesheet" href="https://cdn.rawgit.com/MaayanLab/bioinput/v0.1.0/dist/bioinput.min.css">
  ...
</head>

Secondly, include the JavaScript. Be careful with the order of the projects you're including. jQuery should be first, then typeahead.js, and finally Bioinput. These can also go in your <head></head> tags, however its often best to include them at the bottom of your html body. (Right before </body>) 

          
  ...
  <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/typeahead.js/0.11.1/typeahead.bundle.min.js"></script>
  <script src="https://cdn.rawgit.com/MaayanLab/bioinput/v0.1.0/dist/bioinput.js"></script>
</body>

Usage and Example

The easiest way to use Bioinput is to add two attributes to any html input.

Adding data-role="bioinput" turns a normal input into a Bioinput and adding data-entity-type="assay" would enable assay autocomplete. (Currently "assay", "cell", "gene", "disease", and "organism" are supported.) 

          
<form>
  <div class="form-group assay">
    <label for="assay">Assay Input</label>
    <input id="assay" class="form-control" type="text"
      data-role="bioinput" data-entity-type="assay" />
  </div>
  <div class="form-group cell">
    <label for="cell">Cell Input</label>
    <input id="cell" class="form-control" type="text"
      data-role="bioinput" data-entity-type="cell" />
  </div>
</form>
