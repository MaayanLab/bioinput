Bioinput

View on GitHub Download .zip Download .tar.gz

Bioinput is a JavaScript library built around jQuery, typeahead.js, and the Bootstrap Tags Input libraries. It allows you to quickly and easily implement an autocomplete html input element for various biological entities.

Getting Started

Include Bioinput in your project by downloading the js file or linking via a cdn.

          
            <div class="form-group assay">
              <label for="assay">Assay Input</label>
              <input id="assay" class="form-control" type="text"
                data-role="bioinput" data-entity-type="assay" />
            </div>

Result of calling $('#assay').bioinput('entities');:
